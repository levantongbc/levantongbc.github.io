🗓️ Bắt đầu năm học mới

⚠️ Quy trình an toàn:
1. Xuất backup JSON trước để lưu hồ sơ năm cũ
2. Bấm nút bên dưới — danh sách HS vẫn còn nguyên
3. Cập nhật tổ mới nếu thầy muốn sắp xếp lại

📋 Lấy Apps Script

Dán code này vào Google Sheets → Apps Script → Deploy as Web App → lấy URL dán vào ô trên.

📋 Google Apps Script

1. Vào Google Sheets → Tiện ích mở rộng → Apps Script
2. Dán code bên dưới → Lưu
3. Bấm Deploy → New deployment → Web app
4. Chọn "Anyone" → Deploy → Copy URL → Dán vào tab Gửi email
