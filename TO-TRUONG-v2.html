📤 Bước 2 – Gửi dữ liệu tuần lên Sheets

📋 Lấy code Apps Script

1️⃣Mở Google Sheets → Tiện ích mở rộng → Apps Script
2️⃣Xóa code cũ, dán code bên dưới → Lưu
3️⃣Deploy → New deployment → Web App → Anyone → Deploy
4️⃣Copy URL → Dán vào ô trên → Xong!
